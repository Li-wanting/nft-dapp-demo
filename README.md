# Web3初学：NFT 全流程示例项目（从零到上线）

本仓库提供一个完整的 **NFT 生成 + NFT 铸造** 的全流程示例，包括

- **NFT 合约（nft-create）**
- **DApp 铸造界面（nft-minter）**

为了方便学习，本仓库包含两个版本：

## 📁 项目结构

```markdown
root
├── nft-create/             # 最终完整版本（参考）
├── nft-minter/             # 最终完整版本（参考）
├── initial.zip             # 初始模板，从这里开始教程
├── README.md
└── .gitignore
```

解压 initial.zip 后，你将获得可运行但未完成的模板项目，教程步骤均基于此进行。

可以随时对比 initial.zip 中的代码与此完整版本，了解每一步如何实现。

## 环境要求
- Node.js ≥ 18
- npm 或 pnpm
- MetaMask 浏览器钱包
